# 前言

1978年，自从《C 编程语言》出版问世，计算机世界经历了一场革命。大型计算机变得更大，个人计算机已经有能力和十多年前的主机媲美。在这期间，C 也有了改变，虽然简洁，但它作为 UNIX 操作系统的语言已经传播的远超过它本身了。  
C 语言的日益流行，多年来该语言的变化，以及未参与设计的团队创建的编译器，这些都表明，需要一个比本书第一版提供的更精确、更现代的语言定义。1983年，美国国家标准协会(ANSI)成立了一个委员会，其目标是在保留 C 语言精神的同时，为 C 语言制定一个明确的、独立于机器的定义。结果就是 C 语言的 ANSI 标准。  
在第一版中已经暗示了标准的形式化结构，但并未描述，尤其是结构体赋值和枚举。
这本第二版的 C 编程语言 描述了 ANSI 标准定义的 C。虽然我们已经注意到语言进化的位置，我们选择以独有的新的形势编写。对于大部分来说，  
我们试图维持第一版的简洁。C 并不是一个大的编程语言，
