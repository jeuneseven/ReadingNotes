# 简单句的扩展

* Girls play games. 简单句
* Girls **are playing** games. 改变谓语动词时态
* **Cute** Girls are playing games **happily**. 加入形容词（修饰名词）和副词（修饰动词）
* **Thousands of** cute Girls are playing games **very** happily. 加入数量和副词（修饰副词）
* **After class** thousands of cute Girls are playing games very happily **on the playground**. 加入介词短语，补充说明时间地点

从上到下是加入扩展，是写句子的过程；从下到上去掉扩展，是读句子的过程

## 形容词和副词

### 形容词和副词的用法

#### 形容词的用法

* 主要用于修饰名词
* 可以放在名词前或放在系动词后做表语

#### 副词的用法

* 可以修饰除了名词以外的元素（动词、形容词、副词、句子...）
* 通常情况下，形容词+ly=副词，也有形容词副词同样的情况

### 形容词和副词的三种比较级别

* 最高级 词+est most+词
* 比较级 more+词+(than)
* 原/同级比较：我和你相同 as+比较的内容原型+as + 对象
* 三种比较级专门针对形容词和副词
* 两者之间比较一般是比较级，三者以上是最高级

## 介词短语

### 介词 prep.

* 介词+名词/代词/doing = 介词短语
* 在句子中表示补充说明细节信息，所以是非核心

#### 表示地点的介词

| 介词 | 表达含义 | 注意 | 
| --- | --- | --- | 
| between, among | 在...之间 | 两者之间用1，三者或以上用2 |
| near, by, beside | 在...旁边 |  |
| in front of / behind | 在...之前/之后 |  |
| on, above, over / under, below | 在...上面 / 下面 | on 表示在表面上方（二维空间），above 表示斜上方，over 表示垂直上方 |
| at / in | 在...地点 / 在...里面 | at 表示点，一维空间；in 表示三维空间 |
| around | 在...周围 |  |
| into/out of | 进入...里面/从...出去 | in 表示静态 into 表示动态 |
| onto/off | 到...上面/脱离... | on 表示静态 onto 表示动态 |
| across, through/past | 穿过.../经过... | across 表示在表面穿过，through 表示从里面穿过，past 表示从旁边经过，没有接触 |
| along | 沿着... |  |
| from/to | 来自.../朝着...去 |  |
