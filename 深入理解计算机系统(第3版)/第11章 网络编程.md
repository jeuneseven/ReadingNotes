# 客户端-服务器编程模型

* 网络应用均基于 客户-服务器 模型
* 一个应用由一个服务器进程和一个或多个客户端进程组成
* 客户端、服务器是进程，而非机器或主机
	* 一台主机可同时运行不同的客户端、服务器
	* 客户端、服务器的事务可在同一/不同主机上
	* 无论客户端、服务器如何映射到主机，客户端-服务器模型相同

## 模型的基本操作是事务

	客户端服务器事务并非数据库事务，没有数据库事务的任何特性

1. 客户端需要服务，向服务器发送请求，发起事务
2. 服务器收到请求，解释请求，操作资源
3. 服务器给客户端发送响应，等待下一个请求
4. 客户端收到响应并处理
